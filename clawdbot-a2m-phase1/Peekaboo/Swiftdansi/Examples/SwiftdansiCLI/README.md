# Swiftdansi Demo CLI

Minimal macOS CLI that exercises the Swiftdansi library with Swift 6.2.

## Run
From this folder:

```
swift run SwiftdansiDemo --
```

Pass a markdown file:

```
swift run SwiftdansiDemo -- ../../README.md
```

Options:

```
--theme <default|dim|bright|solarized|monochrome|contrast>
--width <n>
--plain
```
