## Design Notes

**Scale insight:** 260 views in a single lazy-loaded Vite bundle is viable — Vite's code-splitting handles it cleanly (3.89s build, individual chunk sizes 2-60 kB gzipped). The App.tsx navigation nav stays manageable because views are grouped into logical nav sections rather than listed flat.

**Architecture decision:** Keeping Horizon UI as `apps/web-next/` separate from the existing Next.js `apps/web/` was the right call. It lets the two UIs coexist without conflict while Tim and David decide which becomes the primary going forward.
